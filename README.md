# WP + Next.js blog

- clone
- run `cp .env.example .env`
- run `docker-compose up`
- run `cd frontend`
- run `cp .env.local.example .env`
- run `yarn`
- run `yarn dev`
- open `localhost:8000`
- go through wp wizard
- go to settings/permalinks, set post name and save changes
- open `localhost:3000`
